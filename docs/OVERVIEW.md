# Overview

NAMAMI is a local-first habit tracker focused on consistent daily execution.
The product keeps the habit workflow simple, while still offering optional motivation features for long-term adherence.

## Core Idea

- Keep data on-device (IndexedDB)
- Work fully offline after first load
- Track habits with clear completion rules
- Add motivation through optional gamification

## Who It Is For

- People who want habit tracking without account setup
- Users who prefer private, browser-local data
- Users who want one place for check-ins, trends, and rewards

## Product Scope

- Home page for check-ins, heatmaps, stats, and quotes
- Rewards page for badges, reward redemption, and streak freeze
- Settings page for theme, quote category, and global gamification defaults

## Typical Daily Flow

1. Open Home and select today (or another date)
2. Mark habit progress using each habit’s goal type
3. Review streaks and heatmap trends
4. Check Rewards for earned badges or redeem streak freeze

## Principles

- Privacy-first: no account or backend required
- Fast feedback: quick logging from one dashboard
- Transparent scoring: points and streaks are rule-based

## What NAMAMI Is Not

- Not a cloud-sync app
- Not a team collaboration platform
- Not a medical, financial, or compliance tool
